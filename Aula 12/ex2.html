Estilo incorporado e seletores CSS - Seletor de tipo (tags)

Seletores CSS são essenciais para especificar quais elementos HTML serão estilizados.

Eles atuam como padrões que identificam os componentes de uma página ou aplicação que receberão estilos específicos.

Temos diversos tipos de seletores. São eles:

  • Seletores de tipo
  • Seletores id
  • Seletores de classes
  • Seletores universais
